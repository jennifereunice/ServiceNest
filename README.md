# ServiceNest

ServiceNest is a full-stack service management application that allows users to discover services, raise service requests, and track service status through a secure and scalable system.

## 🚀 Features
- User authentication and authorization
- Service listing and request management
- Role-based access (Admin / User)
- RESTful backend APIs
- Responsive frontend interface

## 🛠️ Tech Stack
### Backend
- Java
- Spring Boot
- REST APIs
- Maven
- PostgreSQL

### Frontend
- React
- JavaScript
- HTML, CSS
- Axios
